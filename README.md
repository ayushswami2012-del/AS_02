## 📖 Project Description
This is a fully responsive Stopwatch Web Application built using HTML, CSS, and JavaScript.  
It allows users to accurately measure time intervals with Start, Pause, Reset, and Lap functionality.

The project focuses on improving frontend development skills such as DOM manipulation, event handling, and responsive UI design.

## ✨ Features
- Start / Pause stopwatch functionality  
- Reset stopwatch to zero  
- Lap recording with custom event notes  
- Accurate time tracking (hours, minutes, seconds, milliseconds)  
- Fastest and slowest lap tracking  
- Fully responsive design for mobile, tablet, and desktop  
- Keyboard shortcuts (Space = Start/Pause, L = Lap, R = Reset)  
- Clean and modern UI design  

## 🛠️ Technologies Used
- HTML5 – Structure of the application  
- CSS3 – Styling, animations, and responsive design  
- JavaScript (ES6) – Stopwatch logic, DOM manipulation, and interactivity
